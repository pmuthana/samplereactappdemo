# mfpchtbt-ext-frontend
Repo for source code to MFP Chatbot external, front end code

# Prerequisites
- node v22.14.0
- npm 10.9.2
- Required packages (see Setup)

# Setup
1. Navigate to the frontend directory
2. Install node packages by running `npm install`

# Starting the Frontend Server
You can start the frontend server by navigating to the frontend/ directory and running `npm run dev`.

This will start the server on Port 5173 by default. You may specify an alternative port like so: `npm run dev --port 8060`.
